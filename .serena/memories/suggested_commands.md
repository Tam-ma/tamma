# Suggested Commands

## Development Commands

### Install dependencies
```bash
pnpm install
```

### Build all packages
```bash
pnpm build
```

### Run development mode (all packages in parallel)
```bash
pnpm dev
```

## Testing

### Run all tests
```bash
pnpm test
```

### Run unit tests only
```bash
pnpm test:unit
```

### Run integration tests only
```bash
pnpm test:integration
```

### Run tests with coverage
```bash
pnpm test:coverage
```

## Code Quality

### Run linter
```bash
pnpm lint
```

### Fix lint issues
```bash
pnpm lint:fix
```

### Format code
```bash
pnpm format
```

### Check formatting
```bash
pnpm format:check
```

### Type check
```bash
pnpm typecheck
```

## Database (PostgreSQL)

### Run migrations
```bash
pnpm migrate:latest
```

### Rollback migrations
```bash
pnpm migrate:rollback
```

## Cleanup

### Clean all build artifacts and node_modules
```bash
pnpm clean
```

## System Utilities (Linux)
- `git` - Version control
- `ls` - List files
- `cd` - Change directory
- `grep` - Search text
- `find` - Find files
